# FABulous IHP SG13CMOS5L Test

This repo contains a deliberately small [FABulous](https://github.com/FPGA-Research/FABulous) fabric for testing the flow with the IHP SG13CMOS5L PDK.


To use it with the IHP SG13CMOS5L PDK, go to the [PDK repo](https://github.com/IHP-GmbH/ihp-sg13cmos5l) and follow the usage instructions.
Also update the `FAB_PDK_ROOT` in `.FABulous/.env/` and let it point to where the `dev` branch version of the IHP-Open-PDK repo is located.
